Principais formatações de texto

Negrito / Destaque

Nesta frase usamos um termo em destaque usando a tag B. (Não semântica)

Nesta frase usamos um termo em destaque usando a tag STRONG. (semântica)

Italico / Enfâse

Nesta frase temos um termo em italico usando a tag I. (Não semântico)

Nesta frase temos um termo em italico utilizando a tag EM. (Semântico)

Texto marcado

podemos criar também um texto marcado utilizando a tag MARK

também podemos utilizar o STYLE dentro da tag HEAD para mudar todos os marcadores de texto

Texto grande e pequeno

Neste parágrafo estamos testando letras grandes e letras pequenas.

Texto deletado

Podemos marcar um texto como excluido, para demonstrar que ele deve ser lido, mas não considerado. Utilizamos a tag DEL.

Texto inserido

Podemos também marcar um texto como inserido para dar ênfase e indicar que ele foi adicionado depois. Utilizando a tag INS.
